# 3D Enclosure

The booth structure can be 3D printed or laser cut.

## STL Files

- `station_a.stl`: Left station enclosure
- `station_b.stl`: Right station enclosure
- `center_console.stl`: Raspberry Pi housing

## Design Notes

- Dimensions: 600mm W x 400mm H x 300mm D
- Material: PLA or PETG
- Wall thickness: 3mm
- Ventilation holes for ESP32 heat dissipation

## Alternatives

If 3D printing unavailable:
- Laser cut MDF with acrylic panels
- Wood construction with painted finish

## Assembly

1. Mount ESP32 inside with screws
2. Route cables through grommets
3. Attach button and LED to front panel
4. Secure headphones/mic jacks
